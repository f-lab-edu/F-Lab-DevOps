### 폴더 구조 및 파일 설명
```markdown
url-shortener/
├─ app/                         # Python 애플리케이션 코드 전체
│  ├─ __init__.py               # app 폴더를 Python 패키지로 인식
│  ├─ main.py                   # FastAPI 앱의 시작점 (진입점)
│  │
│  ├─ api/                      # API 엔드포인트 모음
│  │  ├─ __init__.py
│  │  ├─ routes/
│  │  │  ├─ __init__.py
│  │  │  ├─ health.py           # GET /healthz (서버 상태 확인)
│  │  │  └─ url.py              # POST /shorten, GET /{code}
│  │
│  ├─ core/                     # 앱의 핵심 설정/연결 코드
│  │  ├─ __init__.py
│  │  ├─ config.py              # 환경변수 로드 (.env 파일 읽기)
│  │  └─ database.py            # PostgreSQL 연결 & 세션 관리
│  │
│  └─ models/                   # 데이터베이스 테이블 정의
│     ├─ __init__.py
│     └─ url.py                 # Url 테이블 ORM 모델
│
├─ Dockerfile                   # Docker 이미지 빌드 설명서
├─ docker-compose.yml           # API + DB 동시 실행 설정
├─ requirements.txt             # Python 패키지 목록
├─ .env                         # 환경변수 (DB 비밀번호 등)
└─ .gitignore                   # Git 제외 파일 목록
```
